# ComicsBazaar

Hosted at [https://cbls.herokuapp.com].

### Built with
* Frontend: React
* Backend: Node, Koa, Mongoose
* Database: MongoDB

### Features
* Single-page application
* User accounts w/registration and login
* Add/edit/delete listings
